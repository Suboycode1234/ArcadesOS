# ArcadeOS Project Team & Rules

## 🎮 The Project
*   **Name:** ArcadeOS
*   **Purpose:** A gaming-optimized operating system, consisting of:
    1.  A custom 32-bit x86 hobby kernel (located in `C:\Users\sethj\ArcadeOS\`).
    2.  A custom Arch Linux gaming distribution built via `archiso` (located in `C:\Users\sethj\ArcadeOS-distro\`).

1.  **Seth** (The User / Lead Developer): The human developer driving the project.
2.  **Claude** (The Head Architect): The AI that designs, plans, and structures the project architecture.
3.  **ChatGPT** (The Proofreader / Code Reviewer): The AI that reviews documents, proofs scripts, and keeps details clean.
4.  **Antigravity** (The Coding Assistant / Builder): You. You execute code, build binaries, write scripts, compile, run QEMU, and output progress.

## 📜 Team Communication Guidelines
